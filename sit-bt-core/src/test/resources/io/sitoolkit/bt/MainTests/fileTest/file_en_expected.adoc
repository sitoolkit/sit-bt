== Paragraph

== Basic writing style

No special markup is required to describe the paragraph.
It is only described continuously from one line to several lines.

When starting a new paragraph, include at least one blank line.

=== Line break

A plus at the end of a line is considered a line break. +
The ruby color is red. +
Topaz is blue.

[%hardbreaks]
When you specify a hardbreaks for a paragraph, line breaks in the text.
It is displayed as a line feed.
The ruby color is red.
Topaz is blue.

=== Footnote

NOTE: Footnote paragraphs provide additional information.
You can use different types of footnotes depending on the label at the beginning of the paragraph.

TIP: Describes a chip.

IMPORTANT: Describes important matters.

WARNING: Write a warning.

CAUTION: Describes the caution.

== Include File

=== Filename specification

You can include another file by specifying an include statement. +
[] after the file name specifies the tag and number of lines of the included file, and +
You can include only part of a file.
